What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications. It provides a structure for the content of a webpage and defines how elements are displayed on the screen.

What is Semantic HTML?

Semantic HTML refers to using HTML elements that convey meaning and structure to the content they surround. It helps search engines, accessibility tools, and developers understand the purpose and context of different parts of a webpage.

List of Commonly Used Semantic Tags

  • <header>: Represents the introductory content for a page or section
  • <nav>: Defines a section with navigation links
  • <main>: Specifies the main content of a document
  • <article>: Represents a self-contained composition, such as a blog post or news article
  • <section>: Defines a section in a document
  • <aside>: Represents content that is tangentially related to the surrounding content
  • <footer>: Defines the footer for a document or section
